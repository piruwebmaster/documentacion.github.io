    {
   "codProd": "F-BD",
   "codPlan": "001",
   "revPlan": "001",
   "codRamo": "VITR",
   "codMoneda": "RD",
   "codUsr": "externo112",
   "tipoPrima": "LLAMATIVA",
   "tipoPropuesta": "abc",
   "fecIniVigPropuesta": "17/03/2023",
   "fecFinVigPropuesta": "17/03/2024",
   "MostrarComo": null,
   "descripcionBienAsegurado": " nombre descriptivo "
}

    La propiedad Mostrar como tiene un valor por defecto vacio, al enviarse el string "Array" configura la propuesta como multiproducto debiendose enviar la propiedad bienesAsegurados

    {
//...
    "MostrarComo": "Array",
    "bienesAsegurados": [
    {
     //...
        "AnioFabricacion": "2023",
        "descripcionBienAsegurado": "Cámara",
        "DescripcionObjetoAsegurado": "Cámara",
     //...
    }
}

    Si el producto contempla beneficiarios es necesario enviar la propiedad beneficiarios de tipo array de Universal.CoreSeguros.Models.Nivel1.Configuracion.Cotizacion.Dtos.Beneficiario el cuerpo principal de la peticion

    ... 
   "beneficiarios": [
       {
           "numOa": 1,
